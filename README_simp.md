# 中历-公历转换

[English 英文](README.md)  [傳統中文](README_chinese.md)

这是我[公历-农历转换网站](https://ytliu0.github.io/ChineseCalendar/index_simp.html)的 python 版。此软体套件提供一系列的函数可用来做公历-中历转换，可以用来处理较复杂的日历计算。

如果只是想做简单的公历-中历转换，用我的[公历-农历转换网站](https://ytliu0.github.io/ChineseCalendar/index_simp.html)就可以了，简单快捷，本软体套件主要用来处理较复杂的日历计算。

## 安装

用以下 pip 指令安装及更新本软体套件。

`pip install --upgrade ChineseCalendar_py`
