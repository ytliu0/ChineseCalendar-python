# 中曆-公曆轉換

[English 英文](README.md)  [简体中文](README_simp.md)

這是我[公曆-農曆轉換網站](https://ytliu0.github.io/ChineseCalendar/index_chinese.html)的 python 版。此軟體套件提供一系列的函數可用來做公曆-中曆轉換，可以用來處理較複雜的日曆計算。

如果只是想做簡單的公曆-中曆轉換，用我的[公曆-農曆轉換網站](https://ytliu0.github.io/ChineseCalendar/index_chinese.html)就可以了，簡單快捷，本軟體套件主要用來處理較複雜的日曆計算。

## 安裝

用以下 pip 指令安裝及更新本軟體套件。

`pip install --upgrade ChineseCalendar_py`

## 基本使用

[快速入門指南](QuickStart_chinese.ipynb)介紹本軟體提供的基本日曆轉換函數及其基本用法。

## 進階使用

[進階指引](tutorial/traditional_Chinese/tutorial.ipynb)詳述轉日曆換函數的細節。

## 應用

[日曆計算習題](tutorial/traditional_Chinese/calendar_exercises.ipynb)展示如何用本軟體套件的函數編寫程式，處理較複雜的日曆計算。
